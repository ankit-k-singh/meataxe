# Meataxe

Meataxe is a killer collection of Capistrano deployment scripts.

## Installation

Add this line to your application's Gemfile:

```
gem 'meataxe'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install meataxe
```
